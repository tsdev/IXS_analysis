Spec1d

A versatile and rational set of tools to analyse 1d data sets in MATLAB

 

A collection of routines designed to facilitate the analysis of one dimensional (i.e. x, y, error) data sets from the command line in MATLAB.

Data can be loaded into a structure, known as a spec1d object, and then processed with simple one-line commands. Liberal use is made of the operator overload feature of MATLAB5 so that operators such '+', '*' are redefined to work on 1d spectra, including the correct handling of error propogation. Using spec1d it is possible to perform complex data analysis, including non-linear least-squares fitting, in a very efficient manner. As it makes use of the load and function libraries from MFIT/MVIEW, it is completely general, and can be used to analyse data from more or less any source.

One of the main benefits of spec1d is that a complete record of the analysis can be saved in a commend file. Once spec1d has been mastered it is possible to go directly from raw data to publication quality graphics without further processing.

Requires MATLAB 5.1 or higher.
